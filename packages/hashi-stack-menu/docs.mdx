### Example prop usage

To use the `items` prop means providing a lot of JSON data for the component to render. For convenience we have some named exports to assist.

Let's start at the top-level `items` prop.

At this level, the prop data is pretty simple, and is used to render `<NavItem />` components

```javascript
const items = [
  {
    title: 'Browse Products',
    sections: DEFAULT_MENU_SECTIONS,
  },
  {
    title: 'HashiCorp Cloud Platform (HCP)',
    linkUrl: 'https://cloud.hashicorp.com',
  },
  { title: 'About HashiCorp', linkUrl: 'https://www.hashicorp.com/about' },
]
```

When `sections` are provided, a `<NavItem />` will render as a dropdown-style button and expect the appropriate data for its dropdown-activated panel.

Let's explore the named exports we've included to assist in adding `sections`

##### Named export: `DEFAULT_PRODUCT_GROUPS`

Default HashiStackMenu sections are exported to conveniently populate the _Browse Products_ panel

```javascript
export const DEFAULT_MENU_SECTIONS = [
  {
    type: 'product',
    groups: [
      DEFAULT_PRODUCT_GROUPS['infrastructure'],
      DEFAULT_PRODUCT_GROUPS['networking'],
    ],
  },
  {
    type: 'product',
    groups: [
      DEFAULT_PRODUCT_GROUPS['security'],
      DEFAULT_PRODUCT_GROUPS['applications'],
    ],
  },
  {
    type: 'platform',
    groups: [
      {
        title: 'Cloud',
        items: [
          {
            platform: 'HashiCorp Cloud Platform (HCP)',
            url: 'https://cloud.hashicorp.com',

          },

]
```
