## Usage

```css
.wrapper {
  composes: g-grid-container from global;
  composes: g-grid from global;
}

/* Place items within grid columns */
.item {
  grid-column: 1 / -1;

  @media (--medium-up) {
    grid-column: span 6;
  }
}
```
